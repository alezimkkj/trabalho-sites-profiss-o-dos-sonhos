Em qual profissão eu quero chegar

Engenheiro de Dados

Escolhi a Engenharia de Dados como a minha profissão dos sonhos porque sou fascinado pela forma como grandes volumes de informação movem o mundo atual. Como engenheiro de dados, o meu papel será construir as estruturas, pipelines e ecossistemas robustos que transformam dados brutos em insights valiosos para tomadas de decisão em empresas globais.

O que faço para chegar lá?

Atualmente, estou numa jornada intensa de estudos para tornar esse objetivo uma realidade prática. Dedico minhas horas ao aprendizado de linguagens como Python, estruturas de bancos de dados e arquitetura em nuvem, preparando-me de forma constante para as exigências de um mercado altamente valorizado.

Qual meu objetivo?

O meu maior combustível para evoluir nos estudos todos os dias é a perspetiva de conquistar total liberdade geográfica. Pretendo usar o meu conhecimento técnico para trabalhar de forma 100% remota, cruzando fronteiras e resolvendo problemas complexos de infraestrutura digital enquanto exploro novos países e culturas.

"Organizar os dados do mundo para construir a minha própria liberdade, fazendo do planeta o meu escritório."

Importante

Viajar pelo mundo por um tempo não será apenas um período de lazer, mas uma oportunidade de expansão profissional. A Engenharia de Dados é uma das carreiras mais flexíveis do planeta, permitindo-me gerir ecossistemas de informação a partir de qualquer lugar, seja numa grande metrópole ou num chalé isolado nas montanhas.
